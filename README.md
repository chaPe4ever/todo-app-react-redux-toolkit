# TODO App with filtering

## TODO App

This is a simple TODO application built with **React**, **Redux Toolkit** for state management, and **Tailwind CSS** for styling.

### Features

- **Add, remove, and view todos**
- **Filter todos by:**
  - **Search**: Quickly find todos using the search bar
  - **Priority**: Filter by High, Medium, or Low
  - **Status**: Show All, Completed, or Uncompleted todos

### Technologies Used

- **React** (functional components & hooks)
- **Redux Toolkit** for efficient and scalable state management
- **TailwindCSS** for rapid and modern UI styling

### Filtering Capabilities

- **Search Filtering:**  
  Search todos dynamically by their text content with a responsive input field.

- **Priority Filtering:**  
  Filter todos by selectable priority levels: **High**, **Medium**, or **Low**.

- **Status Filtering:**  
  Filter to display **All**, **Completed**, or **Uncompleted** items.
